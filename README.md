# Linked List C Implementation
linked list (singly linked list) c implementation that store character in to linked list

## Installation

```bash
git clone https://github.com/Nanarow/LinkedList-C-Implementation.git
```

## Usage
run **`main.exe`**
```bash
main.exe
```
